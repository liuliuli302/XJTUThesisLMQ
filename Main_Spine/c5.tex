\xchapter{结论与展望}{Conclusions and Future Work}

\xsection{结论}{Conclusions}

本文围绕开放域长视频在免训练摘要场景下面临的全局理解不足、单一评分机制适应性有限以及段级结果难以直接对接标准评测协议等问题，系统开展了基于大语言模型的视频摘要研究，并分别从层级化语义建模与多智能体协同推理两条技术路径进行了方法设计、实现与验证。全文工作的核心目标，不是训练一个依赖大规模标注数据的端到端模型，而是在当前可运行工程主链路基础上，探索如何借助视觉语言模型与大语言模型的语义理解和推理能力，构建具有可解释性、可扩展性与较强开放域适应能力的视频摘要框架。

在第三章中，本文面向传统视频摘要方法存在的语义鸿沟问题，构建了基于层级语义描述与相关性匹配的免训练视频摘要方法。该方法通过视频结构化、帧级与场景级描述聚合、全局语义摘要生成以及多粒度相关性建模，将原始视觉内容逐步映射到统一的文本语义空间，并在此基础上完成帧级重要性评估。该部分工作验证了，将视频摘要任务转化为“语义理解与语义匹配”问题，能够在不依赖额外监督训练的前提下提升摘要结果的主题一致性与可解释性。

在第四章中，本文进一步面向当前项目主执行流程，提出并实现了面向视频摘要的多智能体大语言模型推理框架。该框架以 VideoSummarizationPipeline 为主线，将视频摘要过程划分为视频分段与段级多模态描述生成、Planner 全局规划、多专家协同评分、记忆增强时序建模以及帧级映射与标准协议评测五个相互衔接的阶段。其中，Planner Agent 负责从全部片段描述中推断视频主题、生成全局摘要并动态分配专家权重，Story、Visual、Emotion 与 Information 四类 Expert Agent 则分别从不同维度对片段重要性进行补充判断，最终通过段到帧映射与官方协议评测形成可直接复核的摘要结果。这一框架表明，视频摘要不仅可以被建模为语义匹配问题，也可以被进一步组织为“全局规划 + 多维证据融合 + 时序映射”的协同推理问题。

从实验结果看，当前仓库内已归档的真实评测记录表明，本文所实现的多智能体摘要框架已经能够在 SumMe 与 TVSum 两个公开数据集上形成较稳定的基准表现。其中，DeepSeek 后端在 8 段划分并采用 normalized\_smoothed 评测变体时，在 SumMe 上取得了 $51.89\%$ 的已核验 Mean $F_1$；在 16 段划分并采用 normalized\_raw 评测变体时，在 TVSum 上取得了 $52.25\%$ 的已核验 Mean $F_1$。进一步的设置级消融还表明，分段粒度与评测后处理会对不同数据集产生不同影响：SumMe 更受益于较粗粒度分段与轻量平滑，而 TVSum 更受益于更细粒度的片段建模。结合代表性案例分析可以看到，当前方法能够较好地区分教程类视频中的关键步骤、景观类视频中的代表性视角以及 vlog 类视频中的事件节点，从而体现出一定的跨类型适应能力。

总体而言，本文工作的主要结论可以概括为以下三个方面。第一，在免训练设定下，通过视觉语言模型与大语言模型构建视频的高层语义表示，可以有效缓解传统视觉特征方法对主题理解不足的问题。第二，相较于单一评分器，基于 Planner 与多 Expert Agent 的协同推理机制更有利于在叙事推进、视觉显著性、情绪表达与信息增量之间建立可解释的综合判断。第三，只要将段级语义推理结果进一步映射到标准评测协议要求的帧级表示，并结合统一的评测与聚合接口，training-free 视频摘要方法同样能够形成从方法设计到公开基准验证的完整研究闭环。上述结论共同说明，大语言模型驱动的视频摘要研究不仅具有方法上的可行性，也具有较强的工程落地潜力。

\xsection{展望}{Future Work}

尽管本文已经围绕免训练视频摘要完成了两条技术路径的建模与实验验证，但从当前代码实现、归档结果与实验覆盖范围来看，相关研究仍有进一步拓展空间。首先，当前多智能体主链路中的视频结构化方式仍以按段数切分、固定窗口切分和滑动窗口切分为主，虽然这些机制已经能够支持较稳定的实验流程，但其分段边界主要由预设参数控制，尚未充分利用视频内部的语义变化信息。后续工作可以在保持现有主链路可运行性的前提下，进一步探索更具自适应性的分段策略，使推理单元划分更贴近真实事件边界，从而提升段级描述与后续评分的一致性。

其次，当前框架中的记忆机制仍然属于轻量级上下文注入方式，主要通过拼接前序片段 caption 为后续评分提供历史语境。这一设计在工程上具有实现简单、与现有大语言模型接口兼容性强等优点，但对长程依赖、跨段事件回溯以及复杂叙事链建模的支持仍然有限。未来可以在现有 Planner 与 Expert 协作框架基础上，进一步研究更稳定的记忆组织与检索机制，使模型不仅能够利用最近若干片段的信息，还能够针对当前片段动态检索更具判别价值的历史上下文，从而增强长视频场景下的全局一致性。

再次，从实验验证角度看，当前仓库中已经具备总体性能、设置级消融与代表性案例分析等真实结果，但对于模块级消融、更多后端模型对比、更多分段配置以及更大范围的数据集测试，仍有待形成更系统的实验档案。例如，现有主链路已经支持是否启用记忆机制、不同 LLM 后端以及 5-split 与 50-split 聚合等配置接口，后续可以围绕这些现有开关和评测管线开展更细粒度的对照实验，以更全面地分析各模块设计对摘要性能、鲁棒性和泛化能力的具体影响。

最后，从系统形态看，当前项目已经具备从单视频推理、单视频评测到整数据集聚合分析的完整工具链，但在运行效率、结果可复用性和多模型协同扩展方面仍有继续优化的空间。后续研究可以进一步完善实验配置管理、结果缓存与错误恢复机制，并探索引入更强的多模态描述模型或更细分的专家角色，以便在不破坏当前主链路稳定性的前提下，持续提升系统的表达能力与研究支撑能力。

综上，本文关于视频摘要的研究仍处于持续演进阶段。未来工作将以当前已验证的工程主链路和实验结论为基础，继续围绕更合理的视频结构化方式、更有效的长程上下文建模、更充分的实验验证体系以及更稳定的系统实现展开，从而进一步提升大语言模型驱动视频摘要方法的性能、鲁棒性与实际应用价值。
